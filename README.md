# Saude

Esta página web é dedicada para auxiliar usuários idosos ou de outras etárias sobre saúde e recomendações de uma alimentação saudável de acordo com  nutricionistas.
Pode ser utilizada por ONGS, unidades de saúde públicas e privadas. 

Através desta página o usuário pode ter:

- Recomendação sobre vitaminas, benefícios e em quais alimentos pode encontrar
- Cadastro
- Login
- Cálculo de IMC
